# Ship Thruster
A 2D asteroids style arcade game built with HTML canvas and Javascript. In the game, your goal is to shoot the red circles, and gain poionts as you do. The game gets challanging as you go on!

## How to Play
1. Go to `https://faizy179.github.io/Ship-Thruster/` and press SPACE to begin
2. Press SPACE to fire your lasers and fire your thrusters at the same time. When you let go of the space bar your ship will automatically spin providing you with some aim!
3. Every 5 enemies killed increases the maximum amount of enemies on screen. Don't let the enemies touch you or else it's game over!

## Local Installation
To run this locally proceed with the following steps
1. Clone the repository
2. Go to the 'Ship-Thruster' folder
3. Double click the 'index.html' file to open it in your browser
